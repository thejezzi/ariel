# Content Model

Every document should be one of these:

- **Guide** (task oriented)
- **Concept** (understanding oriented)
- **Tutorial** (step-by-step oriented)
- **Reference** (exact lookup oriented)

## Minimal page contract

Each page should include:

- a clear H1 title
- 1–2 sentence intro
- structured sections (`##`)
- at least one concrete artifact (table, code, or diagram)
