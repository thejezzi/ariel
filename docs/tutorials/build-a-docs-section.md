# Build a Docs Section

This tutorial walks through creating a new docs section.

## Step 1 — create folder

```bash
mkdir -p docs/operations
```

## Step 2 — add ordering

Create `docs/operations/.order`.

```txt
overview.md
runbook.md
```

## Step 3 — add content files

Add markdown files referenced in `.order`.

## Step 4 — verify in sidebar

Restart server and check the new group appears.
